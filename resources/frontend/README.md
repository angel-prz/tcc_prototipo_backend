# angelperez_projeto_dbe2

## Requisitos

- Node.js e npm
- Vite


## Instalação

1. Clone o repositório:

```sh
git clone https://github.com/angel-prz/angelperez_projeto_dfe2.git
cd angelperez_projeto_dfe2
```
2. Instalar as dependências Node:
```sh
npm install
```

3. Copiar o arquivo de ambiente e configure:
```sh
cp .env.example .env
```
4. Iniciar o servidor de desenvolvimento:
```sh
npm run dev
```
